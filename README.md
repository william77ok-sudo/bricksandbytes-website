# Bytes & Bricks website

Professional static website for Bytes and Bricks Studio Ltd

## Domains

- Primary public domain: `https://bytesandbricks.co.uk`
- Cloudflare Pages deployment URL: `https://bricksandbytes-website.pages.dev`
- Support email: `support@bytesandbricks.co.uk`

## Publish

Upload the contents of this folder to the root of the GitHub repository connected to Cloudflare Pages. The site uses root-relative links and is intended to deploy at the root of the Cloudflare Pages project.

## Update

- Main styles: `styles/main.css`
- Mobile navigation and copyright year: `scripts/main.js`
- Horizontal logo: `assets/bytes-and-bricks-logo.webp`
- Compact icon and favicon: `assets/bytes-and-bricks-icon.png`
- Web app manifest: `site.webmanifest`
- Email address appears in `contact/index.html` and the legal pages.

## Before launch

Have the privacy notice, terms and disclaimer reviewed for your actual business, services and data practices. Add the registered office and company number if legally required for your UK company website.
