# Bricks & Bytes Studio website

Professional static website for Bricks & Bytes Studio Ltd.

## Publish

Upload the contents of this folder to the root of the GitHub repository used by GitHub Pages. The site uses root-relative links, so it is intended for a custom domain or a user/organisation Pages site. If it will be published beneath a repository subpath, update the `/...` links to include that repository name.

## Update

- Main styles: `styles/main.css`
- Mobile navigation and copyright year: `scripts/main.js`
- Logo: `assets/bricks-and-bytes-logo.png`
- Email address appears in `contact/index.html` and the legal pages.

## Before launch

Have the privacy notice, terms and disclaimer reviewed for your actual business, services and data practices. Add the registered office and company number if legally required for your UK company website.
